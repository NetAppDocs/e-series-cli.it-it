= Visualizzare la sessione utente dello storage array
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Il `show storageArray userSession` Il comando consente di visualizzare il periodo di timeout della sessione per System Manager.



== Array supportati

Questo comando si applica a un singolo array di storage E2800, E5700, EF600 o EF300. Non funziona sugli storage array E2700 o E5600.



== Ruoli

Per eseguire questo comando su un array di storage E2800, E5700, EF600 o EF300, Ã¨ necessario disporre del ruolo Storage Admin (Amministratore dello storage), Support Admin (Amministratore del supporto) o Storage Monitor (Monitor dello storage).



== Parametri

Nessuno.



== Sintassi

[listing]
----
show storageArray userSession
----

8.41
