= Reimposta la baseline RLS dell'array di archiviazione - CLI SANtricity
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Il `reset storageArray RLSBaseline` Il comando ripristina la linea di base RLS (Read link status) per tutti i dispositivi impostando tutti i conteggi RLS su 0.



== Array supportati

Questo comando si applica a qualsiasi array di storage individuale, inclusi E4000, E2700, E5600, E2800, E5700, EF600, EF300, EF50 e EF80, a condizione che tutti i pacchetti SMcli siano installati.



== Ruoli

Per eseguire questo comando su un array di storage E4000, E2800, E5700, EF600, EF300, EF50 o EF80, devi avere il ruolo Storage Admin.



== Sintassi

[source, cli]
----
reset storageArray RLSBaseline
----


== Parametri

Nessuno.



== Livello minimo del firmware

5.00
