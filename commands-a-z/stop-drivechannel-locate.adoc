= Arresta la localizzazione del canale di azionamento - CLI SANtricity
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Il `stop driveChannel locate` il comando spegne gli indicatori luminosi sui vassoi dell'unità accesi da `start driveChannel locate` comando.



== Array supportati

Questo comando si applica a qualsiasi array di storage individuale, inclusi E4000, E2700, E5600, E2800, E5700, EF600, EF300, EF50 e EF80, a condizione che tutti i pacchetti SMcli siano installati.



== Ruoli

Per eseguire questo comando su un array di storage E4000, E2800, E5700, EF600, EF300, EF50 o EF80, devi avere il ruolo Storage Admin.



== Sintassi

[source, cli]
----
stop driveChannel locate
----


== Parametri

Nessuno.



== Livello minimo del firmware

6.10
