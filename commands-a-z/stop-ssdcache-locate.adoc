= Arresta la localizzazione della cache SSD - CLI SANtricity
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Il `stop ssdCache locate` Il comando spegne le spie dei dischi a stato solido (SSD) che sono stati accesi da `start ssdCache locate` comando.



== Array supportati

Questo comando si applica a qualsiasi array di storage individuale, inclusi E4000, E2700, E5600, E2800, E5700, EF600, EF300, EF50 e EF80, a condizione che tutti i pacchetti SMcli siano installati.



== Ruoli

Per eseguire questo comando su un array di storage E4000, E2800, E5700, EF600, EF300, EF50 o EF80, è necessario disporre del ruolo di Support Admin.



== Sintassi

[source, cli]
----
stop ssdCache locate
----


== Parametri

Nessuno.



== Livello minimo del firmware

7.84

11.80 aggiunge il supporto degli array EF600 e EF300
