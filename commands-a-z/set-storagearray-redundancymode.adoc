= Imposta la modalità di ridondanza dell'array di archiviazione - CLI SANtricity
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Il `set storageArray redundancyMode` command imposta la modalità di ridondanza dell'array di storage su simplex o duplex.



== Array supportati

Questo comando si applica a qualsiasi array di storage individuale, inclusi E4000, E2700, E5600, E2800, E5700, EF600, EF300, EF50 e EF80, a condizione che tutti i pacchetti SMcli siano installati.



== Ruoli

Per eseguire questo comando su un array di storage E4000, E2800, E5700, EF600, EF300, EF50 o EF80, devi avere il ruolo Storage Admin.



== Sintassi

[source, cli]
----
set storageArray redundancyMode=(simplex | duplex)
----


== Parametro

[cols="2*"]
|===
| Parametro | Descrizione 


 a| 
`redundancyMode`
 a| 
Utilizzare `simplex` quando si dispone di un singolo controller. Utilizzare `duplex` quando si dispone di due controller.

|===
