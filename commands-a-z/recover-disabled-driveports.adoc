= Ripristina le porte delle unità disabilitate - SANtricity CLI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Il `recover disabled drivePorts` il comando ripristina le porte disco disabilitate.



== Array supportati

Questo comando si applica a qualsiasi array di storage individuale, inclusi E4000, E2700, E5600, E2800, E5700, EF600, EF300, EF50 e EF80, a condizione che tutti i pacchetti SMcli siano installati.



== Ruoli

Per eseguire questo comando su un array di storage E2800, E5700, EF600, EF300, EF50 o EF80, è necessario disporre del ruolo Storage Admin.



== Sintassi

[source, cli]
----
recover disabled drivePorts;
----
